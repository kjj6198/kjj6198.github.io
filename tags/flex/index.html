高度相同的排版解決方案

在前端的頁面中，我們經常會碰到需要相同高度的排版。最直覺的方法就是將容器裡的所有元素設為 float 或是 inline-block。 float 及 inline-block如果是使用 float 排版，不但要先撐開父元素容器(clearfix)，還要針對子元素設定 margin。所以一旦內容太多，或是高度不足就會跑版。 而且，這樣的排版最大的缺陷就是，必須設定高度。 那，如果不設置高度呢？就算...
